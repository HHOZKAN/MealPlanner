# Meal Planner Application

Une application pour gérer les repas en groupe et partager les coûts des ingrédients.

## Structure du projet

- `backend/` : API Laravel
- `frontend/` : Application Flutter

## Configuration du développement

### Prérequis
- PHP 8.x
- Composer
- Flutter SDK
- Dart SDK
- MySQL/PostgreSQL
